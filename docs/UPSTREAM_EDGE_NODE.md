# Upstream: Holo Edge Node (Docker / HolOS)

This repository ships **only the BroNode desktop app** (Python / Tk, PyInstaller builds).

The **Edge Node container image**, HolOS ISO tooling, and related infrastructure live in the Holo open-source monorepo:

**[github.com/Holo-Host/edgenode](https://github.com/Holo-Host/edgenode)**

BroNode’s default Docker image is `ghcr.io/holo-host/edgenode` — the same image built from that upstream project.
